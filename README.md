# KnxControls

[![Version](https://img.shields.io/badge/Symcon-PHP--Bibliothek-purple.svg)](https://www.symcon.de/service/dokumentation/entwicklerbereich/sdk-tools/sdk-php/)
[![Product](https://img.shields.io/badge/Symcon%20Version-5.0-blue.svg)](https://www.symcon.de/produkt/)

Diese Modulbibliothek stellt verschiedene Erweiterungen bereit, um Geräte mit KNX zu verknüpfen

Folgende Module beinhaltet diese Bibliothek:

- KnxControlsSonos ([Dokumentation](KnxControlsSonos))
- KnxControlsHue ([Dokumentation](KnxControlsHue))

## Voraussetzungen

- IP-Symcon ab Version 5.3

## Installation

### Über Module Control (GitHub URL)
1. Öffnen Sie die IP-Symcon Verwaltungskonsole.
2. Öffnen Sie das "Module Control" (Kern-Instanzen -> Modules).
3. Klicken Sie auf "Hinzufügen" und geben Sie die URL des Repositories ein: `https://github.com/m92axi/SymconModul-KnxControls`

### Manuelle Installation
Kopieren Sie den Ordner `KnxControls` manuell in den `modules` Ordner Ihrer IP-Symcon Installation und starten Sie den Dienst neu.
